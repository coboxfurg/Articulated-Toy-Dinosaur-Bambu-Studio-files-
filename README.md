# Articulated-Toy-Dinosaur-Bambu-Studio-files-
This project demonstrates additive manufacturing of an articulated T-Rex model using a Bambu Lab A1 printer. The modular workflow uses separate sub-assemblies with print-in-place joints. The head and torso connect via a peg-and-socket joint, creating a durable, multi-jointed figure that showcases FDM technology's capability.
